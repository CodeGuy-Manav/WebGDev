} The 'plot' function plots the background of you game, it has these arguments
1]width
2]height
3]x(coordinate of background)
4]y(coordinate of background)
5]colour of background
6](OPTIONAL)bg image url for background
example => plot("100px","100px",0,0,"purple","img.png") will create a rectangle
with the size of 100pxX100px at the toplef(x=0,y=0) of the screen with
the background image "img.png" and background color of purple


The 'ImgSprite' function load an image sprite, it has These arguments
1] Image url
2]Width
3]Height
4]HTML ID you want to asign to the image
5]X
6]Y
example => ImgSprite("img.txt",50,50,"g",0,0) will create an image with
width and height 50,50 and at the topleft(x=0,y=0) of the screen with the id "g"


The 'clear' function clears a surface, it has one argument
1]HTML ID of the lement you want to clear
example => clear("p") will clear the screen


The 'collide' function hecks colission of 2 objects, it has these arguments
1]HTML ID of first sprite
2]HTML ID of secons sprite
example => collide("h","c") will return true if collision has occured in sprites with the id "h" and "c"
and false if not


The 'Text' function is used to display text on the screen,it has these arguments
1]The string you want to display("");
2]OPTIONAL)The x position for text on the screen
3]OPTIONAL)The y position for text on the screen
4](OPTIONAL)A list of additional add-ons.The format is:
1)The first item of the list is "i". if you want italics, you must type "i"
on the index zero of this list
2)The second item of the list is "b". if you want bold text, you must type "b"
on the index one of this list
3)The third item of the list is "font".
if you want to change the default font, you must type the name of the desired font in
index two of this list
example => Text("hi",0,0,["i","b","impact"]) will give and html output => hi


The MoveTo function moves any sprite to the desired coordinates, it has these arguments
1]HTML ID of your sprite
2]X coordinate to move
3]Y coordinate to move


example => MoveTo("h",0,100); will move anything with the ID "h" to x=0 and y=100


The 'flip' function is used to flip a sprite on the x or y axis,it has these arguments
1]HTML ID of the sprite to be flipped
2]An array of directions with [x,y]
example => flip("h"[true,false]) will flip the sprite on the x-axis
example => flip("h"[false,true]) will flip the sprite on the y-axis
example => flip("h"[true,true]) will flip the sprite on the y-axis and x-axis
example => flip("h"[false,false]) will not flip the sprite


The 'show' function if used to show a sprite,it hase this argument
1]HTML ID of the sprite to show
example => show("h") will show the sprite at x=0,y=0(this can be altered by using the MoveTo() function)


The 'hide' function if used to hide a sprite,it hase this argument
1]HTML ID of the sprite to hide
example => hide("h") will hide the sprite


The 'Rotate' function is used to rotate a sprite,it has these arguments
1]HTML ID of sprite to rotate
2]Angle for the sprite to rotate
example => Rotate("h",50) will rotate the sprite with the id "h" at 50 deg
